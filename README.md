Course Enrollment System (Python)
Description

The Course Enrollment System is a console-based Python application designed to manage students, courses, enrollments, grades, and attendance.
It demonstrates the use of object-oriented programming, data validation, and dictionary-based data structures for efficient data management.

This project was created as an educational exercise to practice Python fundamentals, OOP design, and basic Git/GitHub workflow.

Features

Add and manage students

Add and manage courses

Enroll students in courses

Add grades to enrolled courses

Track attendance per course

Input validation for all user data

Efficient lookups using dictionaries

Menu-driven console interface

Technologies Used

Python 3

Standard Python libraries only

Git & GitHub for version control

Project Structure
Course Enrollment system/
│
├── main.py        # Application entry point and menu logic
├── student.py     # Student class and related methods
├── helpers.py     # Input validation helper functions
├── README.md      # Project documentation
└── .gitignore

How to Run the Project

Make sure Python 3 is installed

Clone the repository:

git clone https://github.com/annastatora/Course-Enrollment-system.git


Navigate to the project folder:

cd Course-Enrollment-system


Run the application:

python main.py

Example Functionality

Register a student and assign a unique ID

Create courses using course codes

Enroll students in multiple courses

Add grades and attendance per course

Display students and their enrollment summaries
